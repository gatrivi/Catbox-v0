### Why Catbox vs. Kickbacks?
 * **Honest Take-Rates:** We charge a clear 15% platform fee (scaling down to 3% for ecosystem contributors), giving the rest directly back to you. Kickbacks takes 50%.
 * **No Auditable Fluff:** No closed-source background binaries, no remote executable tracking code, and no complicated blockchain bloat.
 * **Atomic Safety:** Ad text is written using local, atomic .tmp state file-swaps. Your prompts, source code, and variables are completely isolated and never read or sent to our servers.
### Configuration & Control
 * **Toggleable Visuals:** Fully choose where ads appear. Supports standard terminal spinner text injection, a minimal VS Code status bar panel, or a separate sidebar workspace window (disabled by default).
 * **Instant Opt-Out:** Don't like a specific phrase? Toggle it off in your dashboard dashboard.
 * **The Rip-Cord:** Want it completely gone? Run a single, clean command:
```bash
catbox revert

```
Your local system configuration files are restored to their exact pristine state instantly, leaving zero background daemons behind.
